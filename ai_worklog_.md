## 3. OCR 处理

### 遇到的问题
1. **Tesseract 未安装**: 改用 EasyOCR
2. **EasyOCR 模型下载失败**: 改用 PaddleOCR
3. **PaddleOCR 版本兼容**: 降级至 2.7.3
4. **NumPy ABI 冲突**: 降级至 1.26.4
5. **oneDNN 错误**: 设置 FLAGS_use_mkldnn=0

### 解决方案
```python
# 禁用 oneDNN
os.environ['FLAGS_use_mkldnn'] = '0'

# 降级版本
paddlepaddle==2.6.2
paddleocr==2.7.3
numpy==1.26.4
```

## 6. 遇到的问题及解决

| 问题 | 错误信息 | 解决方案 |
|------|----------|----------|
| MinerU 符号链接 | WinError 14007 | 禁用符号链接或用 pdfminer |
| API Key 缺失 | MINERU_API_KEY not set | 创建 .env.example |
| OCR 模型下载 | 网络超时 | 改用离线方案 |
| NumPy 版本 | ABI 版本冲突 | 降级 numpy==1.26.4 |
| PaddleOCR 崩溃 | oneDNN 不支持 | 禁用 oneDNN |
| **定位成功率低** | 3.9% | 扩展关键词库+模糊匹配 |
